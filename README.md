# falcon9-co2-visual
# Gráfico de Emisiones de CO₂ del Falcon 9

Este proyecto es un programa en Python que utiliza Matplotlib para visualizar las emisiones de dióxido de carbono (CO₂) del cohete Falcon 9 por tramos de 10 km hasta los 100 km de altura, llegando hasta la línea de Kármán.

## Cómo funciona
- El programa define los tramos de distancia (10 km, 20 km… hasta 100 km) y las emisiones correspondientes en toneladas.
- A partir de los 70 km, cuando el cohete pasa a su segunda fase, las emisiones disminuyen, reflejando la menor generación de CO₂.
- Se genera un gráfico de línea con títulos y etiquetas que muestran claramente la relación entre distancia y emisiones.

## Características
- Visualización clara de los datos de emisiones por tramo.
- Explicación simple del comportamiento del Falcon 9 durante el lanzamiento.
- Ideal para principiantes que quieren practicar Python y Matplotlib.

## Tecnologías
- Python 3
- Matplotlib

## Notas
Este proyecto es educativo y sirve para entender cómo representar datos científicos mediante gráficos, cómo usar Python para visualizaciones y cómo interpretar la eficiencia de un cohete en diferentes etapas de su vuelo.



# Falcon 9 CO₂ Emissions Graph

This project is a Python program that uses Matplotlib to visualize the carbon dioxide (CO₂) emissions of the Falcon 9 rocket in 10 km increments up to an altitude of 100 km, reaching the Kármán line.

## How it works
- The program defines distance increments (10 km, 20 km… up to 100 km) and the corresponding emissions in tons.

- From 70 km onwards, when the rocket transitions to its second stage, emissions decrease, reflecting the lower CO₂ generation.

- A line graph is generated with titles and labels that clearly show the relationship between distance and emissions.

## Features
- Clear visualization of emissions data for each increment.

- Simple explanation of the Falcon 9's behavior during launch.

- Ideal for beginners who want to practice Python and Matplotlib.

## Technologies
- Python 3
- Matplotlib

## Notes
This project is educational and serves to understand how to represent scientific data using graphs, how to use Python for visualizations, and how to interpret the efficiency of a rocket at different stages of its flight.
